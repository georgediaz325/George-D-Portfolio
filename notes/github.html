How to Create a Repository and Push Code to Profile

  1. Click the + in GitHub and click create new repository
  2. Name the repository what you would like to name your directory and click “Add a README file”
  3. Click the code button and copy the SSH key URL
  4. cd into it the directory you would like to push the new directory into
  5. Then write “git clone” followed by the URL copied from GitHub
  6. Then write “git remote -v” to confirm the link
  7. To add things from terminal to Git write “git add” followed by the file name (eg. git add hello_world.txt)
  8. Type “git status” to check the link on files to Git if its green you did it right
  9. When changes/Updates on a file are made Type (git commit -m “whatever changes made eg. Edit/Add followed by the filename”), text displaying the amount of changes made should appear, try to make it a habit to make commits every time you make substantial changes, or you finish a small goal
  10. Type “git push origin main” to push the changes to GitHub when you are done for the day
  11. Then git status again to make sure everything was done correctly, “nothing to commit, working tree clean” should display
  12. Then reload GitHub to check your work
