HTML Notes

Building Your First Webpage

  • Your homepage file should always be named index.html
  • The HTML file requires a boilerplate, VScode has a shortcut where you can just write ! and it will automatically paste the boilerplate
  • The very first thing at the top of the file is the , this tells our browser what version of HTML our page has been coded in
  • Then the html tag that tells the browser we have used HTML to code our page
  • The head tag allows you to add things that won’t go to the main body of our page like the title or specific instructions that you want your browser to apply to the whole page
  • The title is the title of the browser window
  • The body houses all the information that is displayed in the webpage

Paragraphs and Headings

  • The h tags are used for headings and they can go all the way down to h6 1 being the largest and 6 being the smallest
  • The p tags are used for paragraphs and will add breaks between your text
  • The br tag will let you start your text on a new line, without starting a new paragraph

Bold and Italics

  • The strong tag make your text bold, this tag can be used within paragraphs wrapping around words, this tag tells your Webpage that this text is important
  • The em tag make text italic, this tag can also be used in paragraphs like the strong tag

Lists

  • The ul tag makes unordered lists, for each item in the list the item starts with a li tag
  • ol makes ordered lists, for each item in this lists each item starts with a li tag as well

Links

  • The a tag will add links, the opening tag consists of two parts, a name, and a value, in the case for links you would add href after the a tag then followed by the link in quotes, eg. Click me
  • To make a new page on the same site to be able to use links across, make a new file in the same parent directory , when the files are in the same directory you can simply put the file name as the href value, eg. This is a link to my CSS notes
  • Its best practice to give each page its own directory, for example the about page vs the home page, when linking a page from a different directory just put the directory name infront of the file with ../ leading it, eg Back to Cover Page

Images

  • Using img src=“” you can add images to your webpage either by using a file name or a saved image or a URL, if it is a downloaded image on your computer you can link it like so: img src=“images/dog.jpg” alt =“An Image of a dog”(surrounded by opening and closing tags).
  • You should always include an alt tag, it will add a description in the cases where your image does not load
  • Like with links, using ../ allows you to use files outside of the directory you are currently in, for example, you can use ../ to access your images directory while in your pages directory

Input

  • There are lots of tags for asking for the users input, but the one you will use the most is the input tag. Its a self closing tag and you include two attributes inside it, the id attribute which gives your input tag a unique name, and the type attribute which tells the html file what type of input is being asked. for ex:

    Password: input id = “passwordBox” type =“password”

  • Another attribute you can add is value, which will allow you to write a message inside of the input box or button
  • The type of input type attributes you can use include but are not limited to:
    • text : Creates a box for entering text
    • password : creates a box for entering a password (will present the text as black dots to remain secret)
    • button : creates a clickable button (with JavaScript)
    • checkbox : creates a box that users can tick or untick
