JavaScript Notes

  • In order to write javascript you must add a script element to your html file, put it as the last thing before the closing body tag; ex: *open tag* script src="script.js"*closing tag**open tag*/script*closing tag*
  • console.log() is the command to print something to the developer console in your browser.
  • You can think of variables as simply “storage containers” for data in your code.
  • To create a variable in JavaScript, use the let keyword, you can also use const when it is a variable that will never change, and var which is the old way of declaring variables.
  • To declare a constant (unchanging) variable, use const instead of let
