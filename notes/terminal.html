Terminal Notes

Terminal Commands

  • Pwd: will tell you where you currently are
  • Ls: will open the list of contents of the current directory you are in
  • Ls -F: will tell the terminal to classify the output;

    ex: - A trailing “/“ will indicate that this file is a directory - @ indicates a link - * indicates an executable

  • Man ls: will open the manual where you can see all commands (Q will close the manual)
  • cd (followed by the file name): will change/enter a directory
  • cd: will take you to the home directory
  • cd -: will take you back one directory
  • mkdir: will create a new directory
  • touch: will create a new file
  • rm -i: will remove a file
  • rm -r -i: will remove a directory
  • mv: will rename a file (be careful as it will overwrite any file with the same name);

    ex; mv flie1 file1; will rename the misspelled flie1 and rename it file1

  • to move files to a new directory you use, (using “file1” as an example file etc.);

    mv file1../folder1

  • cp: will copy a file
  • cp -r: will copy a directory
  • to open files in a drop down list, type, open /Users/followed by the users name
